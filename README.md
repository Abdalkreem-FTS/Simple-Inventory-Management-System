# Simple Inventory Management System

A console-based inventory manager with a rich, cool terminal UI powered by [Spectre.Console](https://spectreconsole.net/) Package.

## Features

- **Add** products with name, price, and quantity
- **View** all products
- **Edit** any field of a product
- **Delete** a product by name
- **Search** on a product by name

## Prerequisites

- [Git](https://git-scm.com/install/)
- [.NET 10 SDK](https://dotnet.microsoft.com/download)

## Run

```bash
git clone https://github.com/Abdalkreem-FTS/Simple-Inventory-Management-System.git

cd Simple-Inventory-Management-System/

cd IMS/

dotnet run
```
